Data Quality and Warnings

    Operator Recommendation

    Technical Explanation

    TLE: A Two-Line Element set provides mean orbital elements compatible with SGP4 propagation. It is useful for screening and visualization but does not include covariance.

    SGP4: SGP4 is the standard analytical propagator used with TLEs. It is suitable for TLE-based propagation but should not be confused with high-fidelity orbit determination.

    RIC frame: The RIC frame expresses the secondary object relative to the primary object using radial, in-track, and cross-track components. This is more useful than latitude/longitude for close-approach and RPO geometry.

    Range: The total relative distance between primary and secondary. It is a basic geometric screening metric but not a complete risk metric.

    Radial: The local altitude-like separation. It shows whether the secondary is radially above or below the primary.

    In-track: The along-orbit separation. It is often the dominant component in phasing and conjunction problems.

    Cross-track: The out-of-plane separation. Small cross-track values suggest the objects are locally close to the same orbital plane.

    Range-rate: The time derivative of range. A zero crossing is a useful indicator of local closest approach.

    TCA/POCA: Within this dashboard, TCA/POCA is estimated as the time of minimum propagated range inside the selected window. It is not a substitute for a validated operational CDM.

    Covariance: TLEs do not provide covariance. Without covariance, the dashboard cannot compute a reliable collision probability.
