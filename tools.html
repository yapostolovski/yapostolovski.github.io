TLE Orbit Summary

Paste a satellite Two-Line Element set to derive identifier, epoch, orbit class, period, altitude estimates, eccentricity, drag term, and related orbital metadata.

The tool estimates orbital values from the TLE mean elements. Operator, mission, and live operational status are not looked up from external catalogs.

Chile Overpass Planner

This browser-only planner samples the TLE mean orbit from the current browser time for 72 hours. The map and orbital path use the same longitude/latitude projection. It loads Natural Earth 1:10m country boundaries when available and falls back to simplified coordinate outlines; use mission-grade propagators for operational pointing.

Enter one spot per line as: label, longitude, latitude. The map refreshes automatically after edits.

Enter a TLE and plot the next 3 days.

TLE Conjunction Check

Paste exactly two TLE sets. Choose the UTC analysis window for the orbital data. The checker flags possible conjunctions below 5 km for LEO pairs or 50 km for GEO/high-altitude pairs.

Enter two TLE sets and run the conjunction check.
